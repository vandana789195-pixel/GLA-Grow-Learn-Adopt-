## Image Guidelines

### Technical Specifications
- **Format**: JPG for photos, PNG for graphics with transparency, SVG for icons
- **Resolution**: Minimum 1200px width for full-width images
- **File Size**: Optimize to under 200KB per image for fast loading
- **Accessibility**: Always include descriptive alt-text in HTML

### Style Recommendations
- **Bright and colorful** but not overwhelming
- **Diverse representation** of families, cultures, abilities
- **Authentic moments** rather than overly staged
- **Child-friendly** aesthetic with warm tones
- **High quality** and professional

### Diversity Representation
Ensure images show:
- Various ethnicities and cultural backgrounds
- Different family structures
- Children with disabilities included
- Urban and rural settings
- Different ages (infants through preschool)
- Both mothers and fathers engaged in care

## How to Add Images to Website

1. Save images to this folder (`assets/images/`)
2. Use descriptive filenames (e.g., `family-reading.jpg`)
3. Update HTML files with correct image paths
4. Always add alt-text in the `alt=""` attribute
5. Test images load correctly in browser

Example HTML:
```html
<img src="assets/images/family-reading.jpg" 
     alt="Parent and child reading a colorful picture book together" 
     loading="lazy">
```
